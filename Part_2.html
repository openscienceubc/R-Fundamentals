Getting Started with R

What can you do with R

Lots of things.

R is known for and widely used for statistical computations. But really, it’s a programming language, so there is little that you can’t do with R; you’re not tied to using its statistical packages. You can do text and data mining and analysis, web scraping, you can write your own program from scratch, and much more.

But R is still more than just a programming language and tool for statistical analyses. It’s a community of developers and users. This has several implications.

  • First, there is a large support network online. If you Google a question, you will most certainly find a solution.
  • Second, a lot of add-ons have been created for R, increasing its functionality and consequently, your ability to do novel things with the program.

Today, we’re going to be working almost exclusively with the base R packages, that is, the suite of tools that comes with a basic install. At the end, I’ll point you to a few resources for additional add-ons that have the potential to greatly simplify your life in R.

Lastly, R also has certain features that, as a programming language and as a statistical computational tool, make it very accessible to non programmers. This makes it relatively low barrier, but on the flip side, it makes it very easy to use it poorly.

So, what can you do with R? We’re going to touch on a number of things that you can do with R, without delving into any in too much detail - the goal here is put you in a place where you are comfortable enough to engage in further inquiry and to ask questions that will get you helpful answers.

Because you can do so much with R, no single resource will every be able to answer all of your questions. We’ll also increase the complexity of what we explore with R over the course of the next couple of hours.

Examples

But let’s start with a few simple examples. Try typing the following into your console:

mean(1:256)
## [1] 128.5

What’s happened?

  • You’ve calculated the arithmetic mean of the numbers 1 to 256.
  • This single sequence of numbers is a vector and mean() is a function.

Let’s unpack this a bit though.

Third Now that we have our vector, we can calculate the mean with the mean() function. So, we’ll pass myFirstVector to mean():

mean(myFirstVector)
## [1] 128.5

And there you go. We’ve created a vector by assigning a series of values to a variable and calculated the mean of these values; that is, we can generate data and calculate descriptive statistics on them.

But wait, there’s more

R also allows us to ask questions about our data.

For example, we can look at just the beginning, or just the end of our vector using the head() and tail() functions. So, let’s give that a go:

head(myFirstVector)
## [1] 1 2 3 4 5 6
tail(myFirstVector)
## [1] 251 252 253 254 255 256

We can also inquire about the nature of our vector. We can ask, for example, how long it is with the length() function:

length(myFirstVector)
## [1] 256

We can ask if it is a vector with is.vector():

is.vector(myFirstVector)
## [1] TRUE

We can ask what type of data it contains with typeof()

typeof(myFirstVector)
## [1] "integer"

So what can you do with R?

You can (among other things)

  • generate data and store that data in variables
  • run statistical inquiries on that data
  • look at your data in a variety of ways
  • ask questions about your data

Getting Help in R

As we mentioned, because you can do so much with R, no single resource will every be able to answer all of your questions. R is also, in part because of this, an ongoing learning experience. So, you will need to know how to get help.

As it happens, there is a fair bit of help built right into R. We can find built in help pages, search for specific terms within these help pages, and search for functions and variables.

Finding the help page for a function

This can be done typing ?functionName into your console. For example:

?mean

or

?head

Search for a word in the help pages

While a single ? searches for the help page on a given function, two ?? does a key word search in the documentation. For example:

??plot

??"regression model"

Note

If you did not wrap "regression model" in quotations, it would fail to run. We’ll talk more about using characters in R shortly.

Find variables or functions

The apropos() function searches for variables and functions that contain your search term.

apropos("vector")

We can also search for cuntoins that start or end with specific sets of characters.

To find variable and functions that start with a particular set of letters, like ‘m’ or ‘ma’ we prepend the letters with a ^:

apropos("^m")

apropos("^ma")

Find variable and functions that end in a particular set of letters, like ‘r’ or ‘or’ we append the letters with a $:

apropos("r$")

apropos("or$")

Get examples and demos

Many functions and features in R come with examples and demos on how they might be used. We can often get both examples and demos with the example() and demo() functions. For example:

example(plot) ## Examples using the plot function
demo(graphics) ## Demonstration of the graphics capabilities in R
demo() ## Show a list of all demos available

The help built into R can help you explore, discover, and troubleshoot. However, eventually you’ll need to get into online forums, blogs, and other reading materials to troubleshoot and discover more resources to support your particular area of research.
